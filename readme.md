## Installing Fabric ##

Installing Fabric is easy!

### Source Install ###

Download zip from https://github.com/fabric/fabric

```sh
$ python setup.py install
```

### With Easy Install ### 

```sh
$ easy_install fabric
```
    
### With pip ###
(You should really use pip)

```sh
$ pip install fabric
```
